Display an Under Construction page on your website.

== Description ==

This plugin allow you to create an under construction / maintenance page or message.
3 options are available:
- Display a message
- Display an HTML page
- Redirect to another URL

== Installation ==

1. Upload `wp-undercontruction-mode` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure your Under construction mode in "Under Construction" page
